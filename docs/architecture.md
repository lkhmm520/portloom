# Architecture

The current architecture guide has moved to [Architecture](/en/reference/architecture).

PortLoom's easy deployment runs only PortLoom Server and managed sshd on the public Docker host. Server provides the native HTTPS edge on ports 80/443. Existing Caddy, Nginx, or Nginx Proxy Manager remains an optional advanced integration.
